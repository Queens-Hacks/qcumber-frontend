Known Course Listing Issues

Courses appear in SOLUS Search but not on Qcumber

It has come to our attention that there is a discrepancy on SOLUS between courses found using the search function and courses found using the course catalog area. Namely, some sections ONLY show up in searches. This has been reported to the administration, but no promises have been recieved for fixes in the near future. Unfortunately, this mismatch causes Qcumber to be missing these same courses. So, if you are looking at a (especially first-year) course that you expect to have 2014-2015 sections (e.g. CISC 101), but see none, please check the search function on SOLUS. This problem does not affect the accuracy of the listings present on Qcumber, but it seems to be resulting in the occasional omission.

We apologize for this inconvenience, and are working on ways around the issue, but for now, don't panic if you see something missing. As always, thank you for your support, and happy timetabling Queen's! :)
